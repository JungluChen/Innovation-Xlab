# Innovation-Xlab

Static GitHub Pages site for Project Fibula.

## Files

- `index.html`: main narrative page
- `documents.html`: documents library page
- `assets/styles.css`: shared styles
- `assets/app.js`: client-side interactions and motion
- `assets/fibula-research-poster.pdf`: live poster asset

## Current Placeholder Policy

- Team portraits use text-avatar placeholders
- Logos use text wordmarks
- LinkedIn links remain pending
- Video, report, and slide deck remain visibly marked as pending

## Submission Notes

- No backend or build step is required
- All internal links are relative
- The poster asset is live at `assets/fibula-research-poster.pdf`
- Core story content remains readable even if JavaScript does not load
